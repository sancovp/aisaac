# Blog / Page Updates — reframe existing AIsaac assets toward "AI Operating System"

> Which EXISTING pages and blog posts should be re-angled toward the AIOS frame, and HOW.
> Per item: **current angle → AIOS angle**, plus the exact reframe move. Nothing here edits a file —
> this is the plan. All proposals obey the editorial law: publish one rung above; +1-generalize before
> sharing; ascension-boundary close in the self-transformation register (`free-layer/README.md`,
> `OBVIOUSNESS-PARADOX-framework.md`). The frame to inject is in `positioning-thesis.md`.

**The single reframe move, applied everywhere:** stop leading with the *thing on top* (a plugin, a
skill, an agent, an engine) and lead with **"this is an AI operating system = the folder the AI lives in
+ a frontend; the clever stuff runs on top."** Then let the existing copy be the "on top" detail. We are
not rewriting the assets; we are **renaming the category they sit in** so the market word ("AI OS")
points at them.

## A. Top-level pages (the highest leverage — these set the category)

### 1. `opera.html` — HIGHEST LEVERAGE
- **Current angle:** "OpenClaw is one harness, Paperclip orchestrates a company; OPERA does both + life
  + knowledge graph." A feature-superset comparison ("All of the above + …").
- **AIOS angle:** OPERA is **the open engine that AI Operating Systems ship on** — and the comparison
  table's bottom row should read as a *category jump*, not a feature add: OpenClaw/Paperclip/Claude
  Code/n8n are **apps and harnesses**; OPERA is **the operating system they'd be running *on top of*.**
- **How:** Add one line under the hero: *"A harness runs one agent. An orchestrator runs a company of
  them. An **operating system** is the folder they all live in — the structured place that governs how
  every one of them connects. That's OPERA."* In the comparison table, relabel the column from "What it
  does" framing so the bottom row is "the OS the others run inside," not "more features." Keep it at the
  frame altitude — do **not** add the state-graph / cursor / closure-test internals (that's the withheld
  rung).
