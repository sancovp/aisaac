# MEDICINE — The Offer Construction Framework

## The Principle

"The best medicines are actually *regimes* that cover a whole host of ailments. Likewise, what you are selling is not just simply a product, but a product with a list of features. The more those features are felt, changing the problem situations into grounds for applying solutions and experiencing rejuvenation and fortification, the more proven the system seems. In other words, it walks all obstacles and provides solutions that overcome them. The composite of all of these is simply the bundle, and the actual thing you want them to buy that you want to advertise is the main item in the offer stack. You give them a number of versions of this where they get the core product and have tiers of bonus stacks."

## The Acronym

**M.E.D.I.C.I.N.E.**

**MAKE** the list of obstacles in a journey →
**EDUCATE** by describing them in a way that →
**DESCRIBE** informs how they →
**INFORM** compose in order to →
**COMPOSE** induce the emergence of something others consider →
**INDUCE** →
**NEW** and then →
**EXPOSE** it to them.

## The Process

1. **MAKE** — List every obstacle the prospect faces on their journey. Not your features — their OBSTACLES. What stands between them and the desired identity?

2. **EDUCATE** — Describe those obstacles so they understand them. Name the conditions. "This is lead decay. This is SDR inconsistency. This is routing drag."

3. **DESCRIBE** — Frame each obstacle in a way that informs — not just what it is, but what it DOES to them. "Lead decay costs you $X/mo. SDR inconsistency means 40% of your calls are subpar."

4. **INFORM** — Show how the obstacles COMPOSE together. They're not separate problems — they compound. "Lead decay + slow response + inconsistent follow-up = you're losing 30% of revenue you already paid to acquire."

5. **COMPOSE** — Assemble the solutions that address each obstacle into a bundle. Each solution overcomes one obstacle. The bundle overcomes all of them. This is the regime, not a single pill.

6. **INDUCE** — The composite creates emergence. The bundle together does something that no individual piece does alone. "Voice agent + follow-up + routing + CRM = a system that runs itself. Not three tools — one system."

7. **NEW** — What emerged is something others consider NEW. It's not "I bought three AI tools." It's "I have a frontline revenue system that didn't exist before." The novelty is the COMPOSITE, not the parts.

8. **EXPOSE** — Present it to them. The offer stack with tiers: core product + bonus stacks at each level. Grand slam = everything.

## The Value Equation (Medical Frame)

```
        DESIRED IDENTITY × PROVEN LIKELIHOOD
VALUE = ─────────────────────────────────────
        HOW LONG + HOW MUCH + HOW HARD
```

MEDICINE constructs the numerator. Each obstacle overcome increases desired identity. Each proof point increases proven likelihood. The regime IS the proof — the more features felt, the more proven the system seems.

The core product is the main item you advertise. The bonuses are additional solutions to additional obstacles. Each tier walks more obstacles.
