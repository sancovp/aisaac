## How New Posts Land

When CartON auto-publishes a new post:
1. JourneyBlog template generates the content (AIDA fractal)
2. Post gets tagged with facet + level + book bucket
3. Blog index places it in the right section
4. Category CTA auto-assigned from facet (CAVE→transform, PAIAB→build, SANCTUM→learn)
5. "Belief installed" field ensures every post has a clear belief it installs
