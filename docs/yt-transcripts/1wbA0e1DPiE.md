## Core Pattern (The "Boring Website" Money Loop)

```
1. FIND a generic high-traffic utility (free tool people search for: invoice gen, etc.)
2. NICHE DOWN — don't clone the #1 dominator; pick a specific sub-variant
   (validate the niche with Google Trends + a niche-research blog)
3. BUILD it fast with AI vibe-coding (no tech skills, ~1-5 min)
4. PUBLISH + get a domain
5. MONETIZE with Google AdSense auto-ads (inject the code via the AI)
6. EARN passively as organic traffic accumulates
```

**Why a niche, not a clone:** the generic invoice generator is already "the top dog dominator on Google" — you cannot out-rank it. So you make a more specific version (retainer invoices for service businesses) to capture a narrower, winnable search term.

## Key Evidence / Examples

### The Proof Site (existing "boring website")
- Site type: free online **invoice generator** ("how to generate invoices") — put your logo, from/bill-to/ship-to, generate.
- Monetization: small AdSense display ads placed up and down the page.
- Traffic (via **SimilarWeb**): averaging ~**2 million visitors/month**.
- Revenue (via **AdSense estimator**, "finance in America" category, 2M traffic): ~**$81,000/year**, claimed "completely passively."

### The Build Demo (niche version)
- Niche chosen: **retainer invoices** for **service-based businesses** — found by basic research (a blog on "most popular invoices for service-based businesses").
- Niche validation: **Google Trends** for "retainer invoice" (global, last 5 years) trending up, "doing amazing," rising this year.
- Tool: **Hostinger Horizons** (AI website/web-app builder, "Vibe Coding").
- Prompt used: *"make me an invoice generator website specifically for retainer invoices for service businesses."*
- Time to build: **under 5 minutes** (generation takes ~1-2 min).
- Resulting site: clean retainer-invoice generator — client name/email, payment terms (net 10 / net 30), generate + store invoices, then **publish** to get a domain.

### Monetization Step (AdSense)
- Sign up for **Google AdSense** → Ads section → enable **auto ads** → copy the **AdSense code**.
- Instead of coding, ask the AI builder to inject it: *"Please inject this AdSense code into the header of my website."*
- Hostinger AI adds the code automatically; then **publish**.

## Takeaways

1. **The asset is search-driven traffic on a free utility, monetized by display ads** — not a SaaS, not a sale; ad impressions are the revenue.
2. **Niche-down is the ranking strategy** — you don't beat the incumbent, you pick a winnable sub-term and validate it (Trends + niche research) before building.
3. **AI vibe-coding collapses the build cost to near-zero** — the moat is no longer "can you build it," it's "can you find a niche + get traffic."
4. **Portfolio play** — the $13.99 tier (25 sites) reframes this as building *many* boring sites, spreading the traffic bet.
