## Core Framework: Vibe Coding to Profitable AI Agents

### The Thesis
You do not need to be a developer to build profitable AI agents. "Vibe coding" = describing what you want in plain English, letting AI tools generate the code. The tutorial walks through going from zero to a running, sellable AI agent product.

### Key Principles

1. **Build agents that solve YOUR problems first** — Intimacy with the problem domain is the #1 predictor of product-market fit. Olly's own products (Response AI, Seer) solved problems he experienced in sales/outreach.

2. **Favour workflow over autonomy initially** — Start with a simple, deterministic agent that does ONE thing well. Don't build a general-purpose "autonomous agent." Build a pipeline.

3. **Work bottom-up: one simple agent, then add** — Start with the smallest useful unit. Prove it works. Then layer capabilities.

4. **The SaaS exit model** — Build it, grow it to meaningful MRR, sell it. Not "build and hold forever." SaaS multiples in 2026 for AI products are attractive.
