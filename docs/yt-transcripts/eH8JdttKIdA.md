## TL;DR

Greg Isenberg explains Firecrawl (a web scraping API that returns clean markdown/JSON from any website) and frames it as the "AWS moment for web data." The core thesis: AI agents are blind without clean web data, Firecrawl is the eyes-and-hands layer, and the money is in building hyper-niche vertical SaaS products on top of it — selling the DATA OUTPUT, not the tool. He walks through 7+ startup ideas, a 5-step framework for building data businesses, and a 5-layer AI agent stack.

## Key Frameworks & Models

### 1. Three Eras of AI
- **Era 1 — Chatbots** (2022): Answers questions, limited
- **Era 2 — Copilots** (Cursor, GitHub Copilot): Faster but human still drives
- **Era 3 — Agents** (Claude Code, etc.): AI does the work autonomously, but needs DATA

**Insight:** Each era's ceiling is determined by data quality. Agents without clean web data are "blind."

### 2. The Five-Layer Agent Stack
Every builder needs:
1. **Agent Harness** — Claude Code, Cursor, Codex, IdeaBrowser Pro
2. **Search Layer** — Perplexity MCP, Exa
3. **Web Data Layer** — Firecrawl (scraping, browsing, extraction) ← THIS IS THE EPISODE FOCUS
4. **Ops Brain** — Obsidian, Notion (meeting notes, context storage)
5. **Outbound/Audience Stack** — Instantly, Apollo

### 3. The AWS Analogy for Web Data
- 2006: Wanted a web app → buy servers, manage racks → AWS said "one API call"
- 2026: Want AI to use web data → build scrapers, manage proxies → Firecrawl says "one API call"
- Companies built on AWS became trillion-dollar companies. What gets built on the web data layer?

### 4. Firecrawl's Six Superpowers
1. **Scrape** — one page to clean markdown
2. **Crawl** — entire site automatically
3. **Map** — all URLs on a domain
4. **Search** — Google with full content in one call
5. **Agent** — describe data, it finds it
6. **Browser** — AI controls a real browser (forms, buttons, auth, pagination)

### 5. The Niche Vertical SaaS Framework
**Core thesis:** Take a horizontal SaaS category (SEO tools, job boards, price trackers) and build a hyper-niche version using Firecrawl at a fraction of the cost.

**Why it works:** Constellation Software (~$75B company) built on hundreds of vertical software acquisitions. People pay for specificity. Indeed has 300M listings — nobody wants 300M, they want 50 that matter.

**Pricing:** Incumbents charge $200-$1000/mo for generic tools. Your niche version charges $29-$500/mo for one thing done perfectly for one customer. Margins: 95-99%.

### 6. The 5-Step Data Business Framework
1. **Pick a niche** — What data do people in this industry actually pay for?
2. **Build the scraper** — Firecrawl agent + Python script or Claude Code
3. **Package it** — CSV, dashboard, Slack alert, or API
4. **Sell the output** — $500-$5,000/mo per client (sell data, not the tool)
5. **Automate it** — Schedule it, let it run while you sleep, compound clients

### 7. Seven Startup Ideas
1. **Price Monitoring** (niche: sneaker resale) — alerts on StockX/Goat/eBay, $50-$500/mo
2. **SEO Gap Finder** (niche: dentists only) — competitor analysis + GMB, $200-$500/mo
3. **Job Aggregator** (niche: remote AI/ML only) — monitor 500 career pages, $29/mo premium alerts
4. **AI Research Reports** (niche: crypto token due diligence) — risk scores for VCs, $1K-$5K/mo
5. **Agent in a Box** (niche: real estate comp reports) — pull listings/tax/permits, $300/mo
6. **Review Intelligence** (niche: Amazon FBA seller tracker) — competitor review trends, $99/mo
7. **Lead Gen Service** — 50 company names → enriched CSV, $100-$500/batch, $2 cost

### 8. AI Agents as Employees
Firecrawl posted a job listing for an AI agent employee. Greg's startup idea: build AI agents that companies want to hire.
- Content creator agent: $5K/mo salary
- Customer support agent: $5K/mo salary
- Junior developer agent: $5K/mo salary
