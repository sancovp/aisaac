## Scene 2: The Collapse Pattern (0:15 - 1:15)
**Visual:** [AVATAR: GNOSYS, direct] → [CUT TO: Excalidraw diagram]
**Voiceover:** "You've lived this cycle. Build the foundation. Feel like you understand it. Get excited, start the next layer. Next layer reveals the foundation wasn't actually solid. ## Scene 3: The Four Properties (1:15 - 2:30)
**Visual:** [AVATAR: GNOSYS counting] → [REMOTION-ANIMATION: four-property check]
**Voiceover:** "The OK Stable Signal has four properties. Observable — you can see it working. Not 'I think it works.' You watched it work. Stable — it works repeatedly. Not just once. Not under ideal conditions. Repeatedly. Load-bearing — it holds weight when you build on it. It doesn't just exist — it SUPPORTS the next layer. Independent — it works without you babysitting it. If you have to be there for it to work, it's not stable. All four. 'Worked once' is 30% stable. 'Worked across three contexts' is 80%. 'Independently verified' is 85%. Skip the check, the system collapses under load."
**Remotion animation:** Four checkboxes appearing — Observable, Stable, Load-bearing, Independent. Each gets a demonstration animation, then all four green triggers "OK TO ADVANCE" signal
